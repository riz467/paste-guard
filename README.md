# Paste Guard

AIサービスへのペースト時にセンシティブ情報（APIキー・パスワード・トークン等）を自動検知してマスクする Chrome 拡張機能。

## 対応サイト

- Claude.ai
- ChatGPT (chatgpt.com)
- Gemini (gemini.google.com)
- Microsoft Copilot (copilot.microsoft.com)

## インストール（開発版）

1. chrome://extensions を開く
2. デベロッパーモード ON
3. 「パッケージ化されていない拡張機能を読み込む」→ src フォルダを選択
